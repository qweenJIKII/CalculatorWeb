使い方

基本

  • 数字/演算子をタップして式を作成し、最後に「＝」で計算します。
  • キーボードでも操作可（0-9, + - * / ( ) . ^, Enter, Backspace, Esc）。

関数の表示/非表示

  • ツールバーの「関数」ボタンで、科学関数を折りたたみ/展開できます。
  • 折りたたみ時はボタンが自動で再配置・縮小され、スマホでも「＝」が見切れにくくなります。

角度単位（度/ラジアン）

  • 「度」ボタンで切替（度 ↔ ラジアン）。三角関数 sin, cos, tan に適用されます。

メモリ

  • MC: メモリクリア / MR: 読み出し / M+: 加算 / M-: 減算

編集/関数

  • AC: 全クリア / CE: エントリクリア / ⌫: 1文字削除 / ±: 符号反転 / %: パーセント
  • √, x², 1/x, ^, sin, cos, tan, log, ln, π, e をサポート（関数は折りたたみ対象です）。

共有・コピー

  • 「コピー」で結果をクリップボードへ。「共有」は端末の共有シートを開きます（対応端末のみ）。

メモ（タスク・日記）

  • 「メモ」タブで、タスクだけでなく日記のような複数行の長文も記録できます。
  • Ctrl+Enter で追加。改行は保持されます。チェックで完了、削除ボタンで除去。
  • データは端末内に自動保存（localStorage: calc_memo_tasks）。他端末とは共有されません。

カメラ（QR / OCR）

  • 「カメラ」タブ内で「QRスキャン」「QR作成」「OCR」を切替できます。
  • QRスキャン: カメラ開始→自動検出。距離プリセット/ズーム/ライトに対応。検出結果はコピー/共有/「開く」で外部ブラウザ指定も可能。
  • QR作成: URLのQRコードを生成。既定はアプリのトップ。URLのコピー/共有/PNG保存に対応。生成はオンライン時のみ（api.qrserver.com）。
  • OCR: 画像選択またはカメラ撮影から文字認識。言語は「日本語 / 日本語(縦書き: jpn_vert) / 日本語+英語 / 英語」から選択可。進捗表示あり、結果はコピー/共有可能。

履歴と永続化

  • 直近10件の履歴を保持。DEG/RAD、メモリ、関数表示状態は自動保存されます。

タイマー

  • モードタブで「ポモドーロ / キッチン」を切替可能（選択は自動保存）。
  • ポモドーロ: プリセット（ポモドーロ/短め/じっくり）、作業/短休/長休/ラウンドを分単位で設定。終了時は自動で次のフェーズへ移行します。進捗バー・バイブ・ビープ・通知に対応。Nで次フェーズへスキップ可能。
  • キッチン: 分・秒を指定してカウントダウン。30秒/1分/3分/5分/10分のプリセットあり。終了時にバイブ・ビープ・通知。Nは無効です。
  • キーボード: Space=開始/停止、R=リセット、N=次へ（ポモドーロのみ）。
  • 保存キー: timer_work, timer_short, timer_long, timer_rounds, timer_k_min, timer_k_sec, timer_mode

PWA（ホーム画面に追加・オフライン）

  • HTTPS配信（Netlify）でPWA対応。ホーム画面に追加してオフラインでも利用可能。
  • オフライン時は自動で offline.html にフォールバック。オンライン復帰で自動再読込します。
