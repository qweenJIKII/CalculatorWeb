電卓 説明

使い方

基本

  • 数字/演算子をタップして式を作成し、最後に「＝」で計算します。
  • キーボードでも操作可（0-9, + - * / ( ) . ^, Enter, Backspace, Esc）。

関数の表示/非表示

  • ツールバーの「関数」ボタンで、科学関数を折りたたみ/展開できます。
  • 折りたたみ時はボタンが自動で再配置・縮小され、スマホでも「＝」が見切れにくくなります。

角度単位（度/ラジアン）

  • 「度」ボタンで切替（度 ↔ ラジアン）。三角関数 sin, cos, tan に適用されます。

メモリ

  • MC: メモリクリア / MR: 読み出し / M+: 加算 / M-: 減算

編集/関数

  • AC: 全クリア / CE: エントリクリア / ⌫: 1文字削除 / ±: 符号反転 / %: パーセント
  • √, x², 1/x, ^, sin, cos, tan, log, ln, π, e をサポート（関数は折りたたみ対象です）。

共有・コピー

  • 「コピー」で結果をクリップボードへ。「共有」は端末の共有シートを開きます（対応端末のみ）。

履歴と永続化

  • 直近10件の履歴を保持。DEG/RAD、メモリ、関数表示状態は自動保存されます。

PWA（ホーム画面に追加・オフライン）

  • HTTPS配信（Netlify）でPWA対応。ホーム画面に追加してオフラインでも利用可能。
  • オフライン時は自動で offline.html にフォールバック。オンライン復帰で自動再読込します。
